🤖 ERIC Robotics – Insight.IO Dashboard

A professional robotics control dashboard built for real-time monitoring and control of robotic systems.
Developed using React, Three.js, and Tailwind CSS, this project simulates an advanced robotics interface with modern UI/UX and 3D visualization.

🚀 Features
📹 Camera System
Live video feed interface
HUD-style overlay for telemetry
Clean, minimal robotic UI display
🌐 3D Map Visualization
Interactive point cloud rendering
Built using Three.js
OrbitControls for smooth navigation
📊 Real-Time System Status
Battery level monitoring
Signal strength indicator
Failsafe & system health tracking
🎯 Mission Control Panel
Mission timer tracking
Pause / Stop mission controls
Live mission status updates
🔁 Mode Switching
AUTO / MANUAL mode toggle
Smooth sliding animation
🚨 Emergency System
Industrial-grade emergency stop button
Pulsing alert animation for emphasis
🕹️ Control Interface
WASD joystick-style control system
Drag-based interaction support
📱 Responsive Design
Fully optimized for desktop & mobile
Adaptive layout for different screen sizes
🛠️ Tech Stack
⚛️ React 18 – Frontend framework
⚡ Vite – Fast build tool
🎨 Tailwind CSS – Utility-first styling
🌌 Three.js – 3D point cloud rendering
🎞️ Framer Motion – Smooth animations
🎯 React Icons – UI icon system
📦 Installation & Setup
# Clone the repository
git clone https://github.com/yourusername/eric-robotics-dashboard.git

# Move into project directory
cd eric-robotics-dashboard

# Install dependencies
npm install

# Start development server
npm run dev
